Fancy box exercise

Check out some examples of Fancybox.

  1. Download the Fancybox javascript library from http://fancyapps.com/fancybox/#license (Links to an external site.)  (version 2.1.5 or higher)
  2. Extract (unzip) the contents of the download into your root folder for this class to a new folder named "fancybox". 
  3. Verify that you can see the "source" sub folder at a path similar to "F://{your-csis2000-folder-name}/fancybox/source"
  4. Create a new web page, name it "fancybox.html" and save it to your site root folder on your local F drive. 
  5. Add 3 or more image elements to your "fancybox.html" page.
  6. (For the next few steps you may refer to the image below)
  7. Add a script reference to the latest jQuery library
  8. Include the external stylesheet reference for fancybox that you have just downloaded at "/fancybox/source/jquery.fancybox.css"

  9. Include a script reference to the external file at "/fancybox/source/jquery.fancybox.pack.js"

  10. For each image on your page, add a link (using the tag) to the image file and set each link's class="fancybox".
  11. Add a jQuery ready function that initializes fancybox.
  12. Optionally add any additional functionality using the options found here: http://fancyapps.com/fancybox/#instructions (Links to an external site.)
